NBA Mvp Predictions

Sherzod, Alexis, Will, Mark


This project uses different machine learning models to predict the next NBA MVP based on previous winners and runner ups. We obtain this information by utilizing and training a neural network with statistical data of previous winners. Almost every feasible basketball stat was utilized and fed into a heatmap which assisted us in better understanding the corellations between win share and other stats. Using 3 different models, we created decently accurate model which does gives a decently correct prediction, with Percentage of occurance. On page Data Page & Terms, you can see the result of a linear regression plot on the data.
